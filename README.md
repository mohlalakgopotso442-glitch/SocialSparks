# SocialSparks App
A Kotlin-based Android app helping users maintain social connections by suggesting small interactions based on the time of day.

## Features
Time-based suggestions: Get social "Sparks" for Morning, Mid-morning, Afternoon, afternoon Snack Time, Dinner, After dinner.
Examples: Morning: Send a "Good morning" text to a family member.
User Interface: Simple i put for time of day, displays suggestion, includes Reset button.
Error handling: constructive feedback for invalid inputs.

## How to use
1.Open the app and enter a time of day (e.g Morning, Afternoon).
2.Get a social interaction suggestion.
3.Tap Reset to clear input and try again.

## Tech
Built with: Kotlin, Android studio
Version control: Github
Automation: Github actions for builds

## Purpose 
helps users like Cora simplify social interactions throughout the day.
